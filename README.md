# algorithms.cpp

Implementing algorithms and data structure in C++.

## Setup

You need CMake (sorry for that).

Just do:
```
git clone --depth=1 https://github.com/akvachan/algorithms.cpp.git
cd algorithms.cpp
cmake -B build
cmake --build build --parallel 10
build/Algorithms
```
